# Capitole, subcapitole si sectiuni

## Sinteza

### Sintaxa Markdown

```markdown
# Capitol
## Subcapitol
### Subsubcapitol
#### Sectiune
...
###### Sectiune
```

### Sintaxa HTML

```html
<h1>Capitol</h1>
<h2>Subcapitol</h2>
<h3>SubsubCapitol</h3>
...
<h6>Sectiune</h6>
```

## Nivelul 1: Capitole

### Sintaxa Markdown:

```markdown
# Capitolul 1

In acest capitol ...
```

### Markdown renderizat:

# Capitolul 1

In acest capitol ...

***

### Sintaxa HTML:

```html
<h1>Capitolul 1</h1>
<p>In acest capitol ...</p>
```

### HTML renderizat:

<h1>Capitolul 1</h1>

<p>In acest capitol ...</p>

***
